# Assignment 3 – Using Git with Improved Notebooks and Scripts

This repository contains linear regression analysis of a dataset comparing salary and years of experience. 

## Structure

- "manual" – code written by hand for this assignment
- "ai"– AI-assisted version

Each folder includes:

- Scripts ("linear_regression.py", "linear_regression.r")
- Saved plots ("linear_regression_python_output.png", "linear_regression_r_output.png")

See the README inside "manual" and "ai" for environment setup and commands.
